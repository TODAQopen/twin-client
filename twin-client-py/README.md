## Dev 

### Environment setup

1. Create and use a virtual environment
    1. `virtualenv venv`
    1. `source venv/bin/activate`
1. Install dependencies
    1. `pip install -r requirements.txt -r requirements.dev.txt`

### Testing

To run tests, run the following in the root directory:

```bash
python -m unittest -v
```
